    Нужен локальный сервер

    Это приложение использует ES-модули и PWA. Браузер не даёт загружать их с диска (file://) — это ограничение безопасности (CORS), а не баг MindDump.

    Откройте папку проекта в терминале и выполните одну из команд:

    cd путь\\к\\minddump
npm start

    или без npm:

    npx --yes serve -l 8080 .

    Затем в браузере откройте адрес, который покажет сервер (обычно http://localhost:8080).

    На GitHub Pages приложение уже отдаётся по HTTPS — там достаточно открыть сайт в браузере.
